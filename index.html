Covid-19 Detection using CNN(Edu-Versity)

This project involves using a Convolutional Neural Network (CNN) for detecting COVID-19 from chest X-ray images. The aim of the project is to utilize AI technology to help diagnose COVID-19, providing a quick and effective solution in the fight against the pandemic. The project includes lessons on implementing the CNN model and includes solutions for common errors, with support available through a WhatsApp group. The project teaches the use of dev-ops method and coding standards used in the industry, and covers the principles of effective communication in the AI field.
